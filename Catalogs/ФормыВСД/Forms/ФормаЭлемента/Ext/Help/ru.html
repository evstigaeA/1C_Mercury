Форма элемента справочника "Формы ВСД" предзначена для ввода информации по формам ветеринарно-сопроводительных документов.

Форма отображается при открытии соотвествующего элемента из формы списка одноименного справочника.

Для заполнения предоставлены следующие поля:

1. Код - код элемента справочника

2. Наименование - наименование формы ВСД
