Справочник предназначен для хранения информации о продукции.

Продукция - это номенклатурная позиция, которая участвует в оформлении хозяйственных операций. Продукция является 4-ым уровнем в записи складского журнала.

Элемент справочника может быть добавлен в справочник одним из следующих способов:

·         Введен пользователем вручную при оформлении хозяйственной операции, например инвентаризации;

·         Автоматически при получении входящих ветеринарно-сопроводительных документов (ВСД), в случае, если продукция не найдена в информационной базе;

·         Получен пользователем из ИС «Меркурий» с помощью выполнения команды «Загрузка списка продукции предприятия» формы списка справочника.

Продукцию можно условно разделить на два типа:

·         Зарегистрированная в ИС «Меркурий»;

·         Не зарегистрированная в ИС «Меркурий».

Отличительной особенностью является то, что не зарегистрированная продукция представлена лишь наименованием продукции.

Зарегистрированная продукция закрепляется за предприятием, которое было указано при ее регистрации и ей присваивается уникальный идентификатор «GUID в системе Меркурий».

Операции доступные из меню «Взаимодействие с ИС "Меркурий"» элемента справочника:

Зарегистрировать продукцию в ИС "Меркурий"

Данная команда предназначена для регистрации новой продукции предприятия в ИС "Меркурий". В случае успешного выполнения операции в ИС "Меркурий" будет зарегистрирована новая продукция с присвоенным ей уникальным идентификатором.

Для регистрации необходимо указать минимально необходимый перечень данных:

·         Код (артикул) – артикул производителя, который присвоен производителем продукции;

·         Наименование – наименование, которое присвоено производителем продукции;

·         Подвид продукции – подвид продукции, к которому относится данная продукция;

·         Предприятие – предприятие, за которым будет закреплена данная продукция;

При указании признака «Производится согласно ГОСТ» необходимо указать ГОСТ согласно которому производится данная продукция.

Обновить данные продукции в ИС "Меркурий"

Данная команда предназначена для внесения изменений информации о продукции в ИС «Меркурий». В случае успешного выполнения операции в ИС «Меркурий» будет обновлена справочная информация по данной продукции. Продукция должна быть зарегистрирована в ИС «Меркурий», а так же активной

Обновить данные продукции из ИС "Меркурий"

Данная команда предназначена для получения актуальной справочной информации по выбранной продукции. В случае успешного выполнения операции в информационной базе будет обновлена справочная информация по данной продукции, если она отличается. Продукция должна быть зарегистрирована в ИС «Меркурий».

Операции Зарегистрировать продукцию в ИС "Меркурий" и Обновить данные продукции в ИС "Меркурий" выполняются из-под текущего пользователя.

Внимание! Изменять данные по продукции может только тот пользователь, у которого есть право на обслуживание предприятия указанного в реквизите "Предприятие".
