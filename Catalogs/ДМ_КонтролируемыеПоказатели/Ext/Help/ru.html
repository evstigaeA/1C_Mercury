Каждый элемент справочника соответствует предопределенному регламентному заданию, выполнение которого подлежит особому контролю.

Открыть форму справочника «Контролируемые показатели»  можно из команды меню «НСИ и администрирование \ Монитор состояния системы \ Контролируемые показатели».

Описание   реквизитов справочника:

<Код состояния> (Состояние)

Тип: Число
Код состояния показателя:
        0 - неопределено
        1 - проблем нет (зеленый)
        2 - предупреждение (желтый)
        3- обнаружены проблемы (красный)

<Дата состояния>

Тип: Дата
Дата и время последнего определения кода состояния показателя

<Описание ошибки>

Тип: Строка
Текст описания возникшей ошибки/исключения, если был установлен код состояния выше 1 («предупреждение» или «ошибка»).

Если при выполнении регламентного задания возникла исключительная ситуация, которую возможно обработать с помощью конструкции «Попытка-Исключение-КонецПопытки», то для соответствующего показателя будет установлен код состояния «3» и записан полученный от платформы 1С:Предприятие текст описания ошибки.

Если регламентное задание выполнилось без критических ошибок, то для соответствующего показателя будет установлен код состояния «1» и очищен текст с описанием ошибки.
