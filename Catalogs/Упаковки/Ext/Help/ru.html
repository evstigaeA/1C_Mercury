Данный справочник предназначен для хранения информации по видам упаковки продукции.

Вызывается:

1. через «Главное меню» - «Все функции»

2. из раздела «НСИ и администрирование» группа «Продукция»

Создание элементов справочника происходит при помощи команды «Подобрать из классификатора» на основании данных классификатора, который поставляется вместе с конфигурацией. При использовании команды пользователь может отметить виды упаковки, которые ему необходимы в работе. Команда расположена на командной панели формы списка справочника.

Для обновления информации существующих элементов справочника необходимо использовать команду «Обновить справочник». Команда расположена на командной панели формы списка справочника.
