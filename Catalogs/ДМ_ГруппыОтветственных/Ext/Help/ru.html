Уведомление пользователей о наличии ошибок выполнения контролируемых регламентных заданий.

Для периодического уведомления пользователей о возникших ошибках выполнения регламентных заданий используется отдельное регламентное задание «Мониторинг показателей».

При обнаружении системой с момента последнего запуска регламентного задания «Мониторинг показателей» (но не более чем за последние сутки) появления кода состояния «3» (Ошибка) у одного или нескольких контролируемых показателей, по списку рассылки почтой будет отправлено письмо с уведомлением (пример):

Тема: Уведомление о проблемах обмена с Меркурием - 1С:Управление ветеринарными сертификатами, редакция 2.0

Текст:
!Регламентные задания

Для указания списка получателей рассылки администратор системы должен в справочнике «Группы ответственных» заполнить предопределенный элемент «Уведомления о проблемах с регзаданиями». Доступ к справочнику осуществляется через пункт меню «НСИ и администрирование \ Монитор состояния системы \ Группы ответственных».

В табличной части нужно ввести стоки с именами получателей рассылки и их электронными адресами.

Чтобы система могла отправлять почту администратор должен настроить системную учетную запись электронной почты (<a href=" e1cib/data/Справочник.УчетныеЗаписиЭлектроннойПочты?ref=b0e89c5c8ebed0b011e8214ea5b59c05">). Для открытия формы настройки системной учетной записи почты можно воспользоваться пунктом меню «НСИ и администрирование \ Органайзер \ Настройка системной учетной записи».
