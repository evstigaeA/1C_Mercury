Справочник предназначен для хранения информации о тех хозяйствующих субъектах, с которыми происходят хозяйственные операции.

Хозяйствующий субъект - это лицо, которое самостоятельно организует свою деятельность и несёт полную ответственность за риск, возникающий от его деятельности. В рамках ФГИС "Меркурий" хозяйствующий субъект - это участник экономики, осуществляющий поднадзорную деятельность.

Элемент справочника может быть добавлен в справочник одним из следующих способов:

·        Введен пользователем вручную и отправлен на регистрацию в ИС «Меркурий» с помощью выполнения команды «Взаимодействие с ИС "Меркурий"» -> «Зарегистрировать контрагента в ИС "Меркурий"». Выполняется в том случае, если данный хозяйствующий субъект еще не внесен в ИС «Меркурий»;

·        Автоматически при получении входящих ветеринарно-сопроводительных документов (ВСД), в случае, если Хозяйствующий субъект отправитель не найден в информационной базе;

·        Получен пользователем из ИС «Меркурий» по введенным критериям поиска:

- по указанному ИНН, при окончании ввода ИНН либо выполнив команду «Заполнить реквизиты по ИНН», которая располагается в форме, рядом с полем «ИНН» будет произведен поиск в ИС «Меркурий» и в случае найденного соответствия будут заполнены реквизиты элемента справочника;

- с помощью команды «Расширенный поиск в ИС "Меркурий"». 

Запись элемента возможна только при заполненном реквизите «GUID в системе Меркурий» (означает, что хозяйствующий субъект зарегистрирован в ИС «Меркурий» и ему присвоен уникальный идентификатор). Так же происходит проверка на уникальность ИНН.

Внимание! Хозяйствующий субъект, который используется при выполнении операций с ИС «Меркурий» должен быть зарегистрирован (присвоен «GUID в системе Меркурий»), а так же активным (Признак «Не используется» не установлен).

Операции доступные из меню «Взаимодействие с ИС "Меркурий"» элемента справочника:

Зарегистрировать контрагента в ИС "Меркурий"

Данная команда предназначена для регистрации нового хозяйствующего субъекта в ИС "Меркурий". В случае успешного выполнения операции в ИС "Меркурий" будет зарегистрирован новый хозяйствующий субъект с присвоенным ему уникальным идентификатором.

Для регистрации необходимо указать минимально необходимый перечень данных:

Юридическое лицо:

·         Страна регистрации – страна, в которой зарегистрирован данный хозяйствующий субъект;

·         ИНН – ИНН хозяйствующего субъекта;

·         ОГРН – ОГРН хозяйствующего субъекта;

·         Наименование – краткое наименование хозяйствующего субъекта;

·         Полное наименование – полное наименование хозяйствующего субъекта с указанием ОПФ так, как оно значится в ЕГРЮЛ/ЕГРИП;

·         Форма собственности – организационно-правовая форма хозяйствующего субъекта согласно общероссийскому классификатору организационно-правовых форм (ОКОПФ);

·         Юридический адрес – адрес регистрации хозяйствующего субъекта.

Физическое лицо:

·         Страна регистрации – страна, в которой зарегистрирован данный хозяйствующий субъект;

·         ФИО – фамилия, имя, отчество регистрируемого хозяйствующего субъекта;

·         Юридический адрес – адрес регистрации хозяйствующего субъекта.

Индивидуальный предприниматель:

·         Страна регистрации – страна, в которой зарегистрирован данный хозяйствующий субъект;

·         ФИО – фамилия, имя, отчество регистрируемого хозяйствующего субъекта;

·         Юридический адрес – адрес регистрации хозяйствующего субъекта.

Внимание! При регистрации хозяйствующего субъекта рекомендуется предварительно произвести поиск по ИНН либо с помощью команды «Расширенный поиск в ИС "Меркурий"» для исключения дублирования информации.  

Обновить данные контрагента в ИС "Меркурий"

Данная команда предназначена для внесения изменений по хозяйствующему субъекту в ИС «Меркурий». В случае успешного выполнения операции в ИС «Меркурий» будет обновлена справочная информация по данному субъекту. Хозяйствующий субъект должен быть зарегистрирован в ИС «Меркурий», а так же активным.

Обновить данные контрагента из ИС "Меркурий"

Данная команда предназначена для получения актуальной справочной информации по выбранному хозяйствующему субъекту. В случае успешного выполнения операции в информационной базе будет обновлена справочная информация по данному субъекту, если она отличается. Хозяйствующий субъект должен быть зарегистрирован в ИС «Меркурий».

Операции Зарегистрировать контрагента в ИС "Меркурий" и Обновить данные контрагента в ИС "Меркурий" выполняются из-под текущего пользователя.
