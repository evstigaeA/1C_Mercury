Справочник "Классификатор единиц измерения"

Справочник предназначен для хранения сведений о списке единиц измерения, используемых организацией (штуки, контейнеры, ящики, коробки и т.д.). Единицу измерения нужно указывать, например, при вводе данных в справочник Номенклатура.

Список единиц измерения можно заполнить с помощью Общероссийского классификатора единиц измерения (ОКЕИ), для вызова которого следует использовать кнопку Подобрать из классификатора командной панели формы списка.

Для того чтобы добавить одну или несколько единиц измерения в справочник Классификатор единиц измерения, следует отметить их флагом, а затем нажать на кнопку ОК.
