Данная форма предназначена для поиска зарегистрированного хозяйствующего субъекта в ИС "Меркурий" по указанным реквизитам.

Форма вызывается из элемента справочника "Хозяйствующие субъекты".

Доступные поля для поиска:

·         ИНН – ИНН хозяйствующего субъекта;

·         Наименование – наименование хозяйствующего субъекта, для физического лица ФИО;

·         ОГРН - ОГРН хозяйствующего субъекта.

Если в элементе справочника уже есть заполненные реквизиты, то они автоматически заполнятся на форме. После указания информации, по которой будет осуществлен поиск необходимо нажать кнопку «Найти» и будет произведен поиск в ИС «Меркурий». В результате будет заполнена табличная часть данными хозяйствующих субъектов, которые попали под критерии поиска. Далее в списке необходимо выбрать нужного хозяйствующего субъекта и нажать кнопку «Выбрать» либо двойным кликом по строке табличной части – реквизиты элемента справочник будут заполнены полученными данными.
