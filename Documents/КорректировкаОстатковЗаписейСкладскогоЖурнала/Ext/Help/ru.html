Документ предназначен для ручного изменения остатков в разрезе записей складского журнала в системе.

В шапке документа необходимо указать Организацию (реквизит "Хоз. субъект" заполнится автоматически по связи с организацией), а так же предприятие по которому требуется изменить данные.

С помощью команды «Подбор» заполнить табличную часть записями складского журнала, по которым необходимо скорректировать остатки. В форме подбора в колонках «Количество» и «Количество упаковок» отображается количество по данным системы.

Табличная часть состоит из следующих колонок:

·         «Запись складского журнала» - запись складского журнала, по которой будет выполнена корректировка.

·         «Номер записи журнала» - номер записи складского журнала.

·         «Единица измерения» - единица измерения указанная в записи складского журнала.

·         «Количество остаток» - остаток по данной записи складского журнала в системе.

·         «Количество упаковок остаток» - остаток упаковок по данной записи складского журнала в системе.

·         «Количество» - количество, которое необходимо отразить как остаток в системе.

·         «Количество упаковок» - количество, которое необходимо отразить как остаток в системе.

При проведении документа на остатках будет числиться количество, указанное в колонках «Количество» и «Количество упаковок» соответственно.

Внимание! Данный документ предназначен только для корректировки записей складского журнала в системе.
