Форма документа производственных операций предназначена для создания/редактирования регистрации операций по транспортировке определенной продукции и создания транспортного ВСД.

При выполнении операции возможны следующие сценарии:

1. В транспортной партии используется ранее выработанная на данном предприятии партия продукции:

1.1. Выработанная продукция с указанной записи журнала была полностью израсходована (отгружена), т.е. объём продукции для записи журнала после списания стал равен нулю.

§  Происходит списание объёмов с одной или нескольких записей журнала.

§  Одна или несколько записей журнала вырабатываемой продукции, объем которых израсходован, переходит в категорию "оформленных";

§  Оформляется ВСД на каждое наименование отгружаемой продукции;

§  Происходит гашение производственных сертификатов, объем которых был израсходован.

 

1.2. Выработанная продукция с указанной записи журнала не была полностью израсходована, т.е. объём продукции для записи журнала после списания – больше нуля.

§  Происходит списание объёмов с одной или нескольких записей журнала.

§  Оформляется ВСД на каждое наименование отгружаемой продукции.

2. В транспортной партии используется одна или несколько партий продукции из входного журнала:

2.1. Входящая продукция с указанной записи журнала была полностью израсходована (отгружена), т.е. объём продукции для записи журнала после списания стал равен нулю.

§  Происходит списание объёмов с одной или нескольких записей журнала.

§  Одна или несколько записей журнала входной продукции, объем которых израсходован, переходит в категорию "оформленных".

§  Оформляется ВСД на каждое наименование отгружаемой продукции.

2.2. Входящая продукция с указанной записи журнала не была полностью израсходована, т.е. объём продукции для записи журнала после списания – больше нуля.

§  Происходит списание объёмов с одной или нескольких записей журнала.

§  Оформляется ВСД на каждое наименование отгружаемой продукции.

 

Для создания документа необходимо заполнить следующие реквизиты

Для создания документа необходимо заполнить следующие реквизиты:

·         Шапка документа:

o   Дата – дата создания документа;

o   Серия ТТН – указывается серия товарно-транспортной накладной;

o   Дата ТТН – дата создания накладной;

o   Номер ТТН - указывается номер товарно-транспортной накладной;

o   Тип ТТН – указывается тип накладной;

o   Хоз. субъект – элемент выбирается из справочника «Хозяйствующие субъекты»;

o   Предприятие – элемент выбирается из справочника «Предприятия»;

·      Табличная часть «Продукция»: в этой части выполняется подбор продукции с помощью кнопки «Подбор», далее дважды нужно кликнуть по позиции продукции и нажать на кнопку «Перенести в документ».

·         Табличная часть «Транспортировка»:

o   Назначение груза – элемент выбирается из списка: «Назначения грузов»;

o   Транспорт - элемент выбирается из списка: «Транспорт»;

o   Посредник - элемент выбирается из справочника «Хозяйствующие субъекты» (если осуществлялась перегрузка продукции при транспортировке);

o   Способ хранения при перевозке – элемент выбирается из списка «Способы хранения при перевозке»;

o   Пункт перегрузки – элемент добавляется при помощи кнопки «добавить» из списка «Транспорт» (если осуществлялась перегрузка продукции при транспортировке);

После заполнения реквизитов нужно нажать на кнопку «Записать» для записи данных в документ, а после нажать на кнопку «Сформировать заявку», чтобы документ получил статус «Новая заявка». После получения заявки Системой документ перейдет в статус «ID заявки получен». Когда Система обработает заявку,  документ перейдет в статус «Заявка успешно обработана», при этом табличная часть: «ВСД» будет заполнена автоматически.
