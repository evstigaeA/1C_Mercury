Документ "Аннулирование ВСД" предназначен для оформления операции "Аннулирования ветеринарно-сопроводительных документов (ВСД)". Если ВСД содержит ошибки, либо фактические сведения отличаются от того, что указано в ВСД, то такой документ может быть аннулирован, то есть такой документ становится недействительным. 

В документе все данные за исключением "Причина аннулирования ВСД" и "Ответственный" заполняются исходя из данных аннулируемого ВСД и не могут отличаться от исходного. Документ не предназначен для ручного ввода и оформляется автоматически на основании документов "ВСД" или "Транспортная операция".

При успешном выполнении операции (документ переходит в статус "Заявка успешно обработана") объем продукции (количество, количество упаковок) на которую был выдан ВСД ставится обратно на приход, а статус самого ВСД переходит в "Аннулирован (не действующий более сертификат)".

Аннулирование возможно только для транспортных ВСД сформированных на основании документа "Транспортная операция", поэтому для корректного выполнения операции необходимо выполнение следующих условий:

1. ВСД должен быть с типом "Транспортный ВСД";

2. ВСД должен быть в статусе "Оформлен (действующий сертификат, по которому разрешено совершать транзакцию с грузом)";

3. Транспортная операция на основании которой был выписан ВСД должна быть в статусе "Заявка успешно обработана".

Офомление документа возможно по команде "Аннулировать" из документа "Ветеринарно-сопроводительные документы (ВСД)" или "Транспортные операции", а так же из одноименных журналов. Формировать можно только по одному документу. Результатом выполнении команды является созданный документ заполненый необходимыми данными. Для регистрации данных в ИС "Меркурий" необходимо выполнить команду "Сформировать заявку".

При выполнении команды "Аннулировать" из документа "ВСД" документ создается сразу, из документа "Транспортные операции" вначале создается документ "Групповое аннулирование ВСД" .
