Данный документ предназначен для отражения возможных несоответствий об объёмах в складском журнале продукции системы Меркурий и фактических объёмов продукции на складе предприятия, выявленных по результатам проведенной инвентаризации. Для документа возможны два вида действия:

1. Изменение, предназначен для отражение несоответсвия в количественном выражении.

2. Добавление, предназначен для ввода новых данных.

Документ можно вызвать двумя способами:

1. Через "Главное меню" - "Все функции"

2. С рабочего стола "Управление ветеринарными сертификатами" из группы "Документы"
