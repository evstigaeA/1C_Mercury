Документ "Групповое аннулирование ВСД" предназначен для формирования документов "Аннулирование ВСД" на основании данных "Транспортной операции". Документ позволяет подобрать все "ВСД" оформленные по данной операции удовлетворяющих условиям документа "Аннулирование ВСД". В табличной части документа пользователь может либо оставить весь список документов либо удалить те строки в которых ВСД не подлежат аннулированию.

Команда "Оформить аннулирование ВСД" предназначена для формирования документов "Аннулирование ВСД" по каждой строке.

Команда "Заполнить причину" предназначена для заполнения одинаковым значением для всех строк реквизита "Причина аннулирования ВСД". Можно откорректировать вручную.

При проведении документа документы "Аннулирование ВСД", которые сформированы будут подготовлены для отправки в ИС "Меркурий".
