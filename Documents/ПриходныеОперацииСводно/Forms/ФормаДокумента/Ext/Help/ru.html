Форма документа предназначена для создания/редактирования операций по поступлению и возврату групповой партии продукции.

Для создания документа необходимо заполнить следующие реквизиты:

·         Шапка документа:

o   Дата – дата создания документа;

o   Хоз. субъект отправитель – элемент выбирается из справочника «Хозяйствующие субъекты»;

o   Хоз. субъект получатель – элемент выбирается из справочника «Хозяйствующие субъекты»;

o   Предприятие отправитель – элемент выбирается из справочника «Предприятия»;

o   Предприятие получатель – элемент выбирается из справочника «Предприятия».

 

«Табличная часть»:

o   ВСД – выбирается элемент (действующий сертификат, транспортный ВСД) из списка «ВСД» и добавляется при помощи нажатия кнопки «Добавить»;

o   Статус ВСД – поле заполняется автоматически после добавления «ВСД»;

o   Приходная операция – поле заполняется автоматически после последовательного нажатия кнопок «Записать» и «Сформировать приходные операции»

После заполнения реквизитов нужно нажать на кнопку «Провести» для обработки заявки Системой, после чего документ перейдет в статус «Заявка успешно обработана».

Примечание – Если ВСД был создан пользователем (Бумажный ВСД), то в поле «Приходная операция» нужно будет заполнить вручную аналогично п.5.4.4 вкладку «Контроль» и реквизиты  в табличной части «Общие сведения»: «Серия ТТН»; «Тип ТТН», «Номер ВСД», «Дата ВСД». Далее нужно нажать на кнопку «Записать» и «Сформировать заявку» и подождать когда заявка будет успешно обработана Системой
