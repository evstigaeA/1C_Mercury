Форма документа ВСД предназначена для создания/редактирования информации по сертификации поднадзорных госветнадзору грузов.

Для создания бумажного ВСД пользователю необходимо заполнить следующие реквизиты:

·         «Общие сведения»

o   Тип документа – выбирается «Бумажный»;

o   Форма ВСД – выбирается элемент из списка «Формы ВСД»;

o   Тип ВСД – выбирается элемент из списка «Типы ВСД»;

o   Хоз. субъект отправитель – элемент выбирается из справочника «Хозяйствующие субъекты»;

o   Хоз. субъект получатель – элемент выбирается из справочника «Хозяйствующие субъекты»;

o   Предприятие отправитель – элемент выбирается из справочника «Предприятия»;

o   Предприятие получатель – элемент выбирается из справочника «Предприятия».

·         «Вет. данные»

o    Благополучие местности – выбирается элемент из справочника «Благополучие местности».

 

·         «Продукция»

o  Тип/Вид продукции – элементы выбираются из справочников: «Типы продукции» и «Виды продукции»;

o  Подвид продукции/ Продукция - элементы выбираются из справочников: «Подвиды продукции» и «Продукция»;

o  Ед. изм./ Количество - элемент выбирается из справочника «Единицы измерения»/количество вводится вручную;

o  Упаковка/ Количество упаковок - элемент выбирается из справочника «Упаковки»/количество упаковок вводится вручную;

o   Дата выработки начало/ конец – элементы вводится вручную;

o   Дата окончания срока годности – элементы вводится вручную;

o   Маркировка/ Некачественный груз – элементы вводятся вручную при необходимости;

o   Скоропортящаяся – поле заполняется вручную при необходимости.

o   Некачественный груз – поле заполняется вручную при необходимости.

·           «Транспортировка»

o   Назначение груза – элемент выбирается из списка: «Назначения грузов»;

o   Транспорт - элемент выбирается из справочника: «Транспорт»;

o   Посредник - элемент выбирается из справочника «Хозяйствующие субъекты» (если осуществлялась перегрузка продукции при транспортировке);

o   Способ хранения при перевозке – элемент выбирается из списка «Способы хранения при перевозке»;

 

Пункт перегрузки – элемент добавляется при помощи кнопки «добавить» из списка «Транспорт» (если осуществлялась перегрузка продукции при транспортировке).

После заполнения реквизитов нужно нажать на кнопку «Записать» для записи данных в документ, а после нажать на кнопку «Оформить», чтобы ВСД был сформирован.
