С помощью данного документа задается перечень предприятий, которые может обслуживать ответственное лицо.

Одно предприятие может обслуживаться несколькими ответственными лицами.

Для назначения предприятия ответственному лицу необходимо добавить строку в табличную часть. В строке выбрать обслуживаемое предприятие и в поле "Операция" выбрать значение "Назначение".
Если ответственное лицо прекращает обслуживать предприятие, то необходимо добавить строку в табличную часть. В строке выбрать предприятие, которое было назначено ранее, и в поле "Операция" выбрать значение "Освобождение".

Для выбора пользователя, реквизиты подключения которого, будут использованы для загрузки ВСД обслуживаемого предприятия необходимо установить флаг "Использовать для загрузки ВСД".

Влияет на отборы:

- Предприятие-отправитель - Транспортная операция
- Предприятие-получатель - ВСД(бумажный), Приходная операция(сводно)
- Предприятие - Инвентаризация, Производственная операция, Ввод начальных остатков
- Предприятие - Групповое аннулирование ВСД, Аннулирование ВСД

Документ является регистратором для регистра сведений "Обслуживаемые предприятия". На основании данных регистра происходит автоматическая загрузка входящих ВСД/обновление статусов ВСД.

Так как реальный список обслуживаемых предприятий задается администратором в ИС "Меркурий", создание списка обслуживаемых предприятий в УВС не гарантирует, что отправленная заявка от лица пользователя по предприятию, которое задано в УВС, вернется со статусом отличным от "Заявка отклонена". Список обслуживаемых предприятий является вспомогательным.

Документ можно вызвать двумя способами:

1. Через "Главное меню" - "Все функции"
2. Раздел "НСИ и администрирование" - "Настройка параметров системы"
