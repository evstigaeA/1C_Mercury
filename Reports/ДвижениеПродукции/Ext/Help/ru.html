Данный отчет предоставляет возможность получения наглядной информации по движениям продукции с указанием требуемых отборов

Для формирования отчета «Движения продукции» пользователь должен заполнить необходимые реквизиты, при этом в «пустом квадратике» около реквизита нужно поставить «галочку»:

o   Период отчета – поле заполняется вручную;

o   Предприятие – элемент выбирается из справочника «Предприятия»;

o   Тип продукции – элемент выбирается из справочника «Типы продукции»;

o   Вид продукции – элемент выбирается из справочника «Виды продукции»;

o   Подвид продукции – элемент выбирается из справочника «Подвиды продукции».

 

После заполнения реквизитов нужно нажать на кнопку «Сформировать» для формирования нужного пользователю отчета

Отчет можно открыть следующими способами:

1. Через "Главное меню" - "Все функции"

2. С рабочего стола "Управление ветеринарными сертификатами" из группы "Отчеты"
