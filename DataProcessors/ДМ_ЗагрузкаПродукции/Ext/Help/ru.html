Обрабатываются все листы, на каждом листе начальная строка = 2.

 Основной задачей обработки есть поиск продукции по указанному GUID. Если элемент не найден, он создается. Иначе перезаполняются ряд полей, указанных в файле: артикул,наименование, тип,вид и подвид продукции. Если вид и подвид не находятся по указанным GUID, элементы создаются в соответствии с иерархией ТипПродукции --> ВидПродукции --> Подвид продукции-->  Продукция.

    Предполагается, что периодически выполняется регламентное задание "Актуализация справочника "Продукция", который по указанным GUID находит в базе Меркурия и обновляет по ее данным элементы продукции в базе 1С.
