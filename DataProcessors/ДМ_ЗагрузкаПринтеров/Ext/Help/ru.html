Обработка выполняет полную перезапись содержимого регистра сведений "Принтеры" данными, загружаемыми из файла Excel.

Колонки в файле с данными для загрузки:

Наименование принтера Plant Основной принтер

 

Загрузка данных выполняется со всех листов начиная с 2-й строки первого листа.

В поле "Имя файла с данными" необходимо ввести полный путь к файлу Excel с данными для загрузки.

Если поле "Имя файла протокола загрузки" не заполнено, то все сообщения об ошибках буду выводится на экран пользователю в окне сообщений.

Если поле "Имя файла протокола загрузки" заполнено, то вместо вывода на экран сообщения будут перенаправлены в текстовый файл, полный путь к которому казан в этом поле. Если такой файл уже существует, то он будет перезаписан, а все прежнее содержимое файла будет удалено.
