Данная обработка предназначена для получения списка продукции, которая зарегистрирована (закреплена) за определенным предприятием-производителем в ИС «Меркурий», т.е. данной продукции присвоен уникальный идентификатор.

Для получения списка продукции на форме необходимо указать предприятие, по которому необходимо получить список продукции, а так же подвид продукции. В случае, если подвид продукции не указан, то будет получен полный перечень продукции зарегистрированной за указанным предприятием.

Поиск в ИС «Меркурий» происходит после того как пользователь нажмет кнопку «Найти». В случае успешного выполнения операции табличная часть будет заполнена полученным списком продукции. Если в процессе получения и обработки информации возникнут ошибки, то в поле «Сообщение» будет выведено описание ошибки иначе сообщение «Операция выполнена успешно». Если по полученной информации в информационной базе будет найдено соответствие, то в строке будет заполнено поле «Продукция», т.е. это обозначает, что данная позиция уже есть в информационной базе.

Для загрузки продукции в информационную базу необходимо отметить те позиции, которые необходимо загрузить и нажать кнопку «Загрузить выбранные». Если отмеченная продукция уже существует в информационной базе, то по ней информация будет обновлена, иначе создан новый элемент.
